Relative Unit

  • %

  • em
    1. Relative to parent element font size
    2. For example if body is parent by default font-size : 16px;
    3. 1em == 16px and 2em == (16*2)px (only if you haven't change the font-size)
  • rem
    1. Relative to root(HTML)
    2. By default for root font-size : 16px;
    3. 1rem == 16px and 2rem == (16*2)px
    4. Changing the font-size of parent container doesn't effect on it
